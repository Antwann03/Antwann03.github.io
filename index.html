Software-Defined Radio Digital Communication System

Built an end-to-end digital communication system using a USRP N210 Software-Defined Radio to explore real-world modulation and signal processing. The project evaluates system performance under different modulation schemes and demonstrates practical SDR pipeline design using GNU Radio.

  • Designed filtering, synchronization, and demodulation blocks
  • Analyzed signal distortion in time and frequency domains
  • Improved link reliability through debugging and tuning
  • Validated performance across multiple modulation schemes

The system uses GNU Radio to construct a full SDR pipeline including modulation, filtering, synchronization, and demodulation. Performance was evaluated using time-domain and spectral analysis, bridging theoretical communication concepts with practical RF experimentation.

Electrical Resistivity Tomography (ERT) System

Developed and improved an Arduino-based Electrical Resistivity Tomography (ERT) system for geophysical sensing. The project automates multi-electrode measurements to eliminate manual switching and increase experimental accuracy.

  • Designed a custom multi-channel PCB in KiCad using 16×1 multiplexers
  • Automated electrode switching to enable continuous measurements
  • Built and tested laboratory-scale sensing hardware
  • Collected resistivity data under varying environmental conditions
  • Validated system performance and measurement repeatability

The system integrates Arduino-based control electronics with a custom switching board to perform automated resistivity measurements. This work combines PCB design, embedded programming, and experimental validation to create a scalable geophysical instrument platform.

L.O.G. (Lethal Offensive Gear) Project

Developed a modular robotic combat chassis inspired by battle bots. The robot is remotely controlled via a web interface hosted on an ESP32 microcontroller.

  • Designed and built the modular chassis using Fusion 360 and physical prototyping.
  • Implemented an ESP32-based Wi-Fi web interface for real-time control.
  • Developed motor driver and power circuits to maintain stable performance under dynamic loads.
  • Integrated peripherals such as GPIO, PWM, and SPIFFS for robot control and UI hosting.
  • Iteratively improved drivetrain and chassis for structural stability and manufacturability.

The ESP32 runs a lightweight HTTP server, serving a custom web interface. Motors are driven through H-bridge inputs controlled via GPIO, with PWM for precise speed control. Images and UI assets are served from the SPIFFS filesystem. The system demonstrates real-time wireless control using Wi-Fi SoftAP mode, enabling operation without a router.

Intersatellite Optical Communication (ISOC) Project (Capstone/Team Project)

Project Overview:
With the adoption of optical communications, the field of communications is rapidly evolving. Inspired by Starlink, this project designs a free-space optical communication system for Low Earth Orbit inter-satellite links. The system calculates the angle of arrival of a laser, acquires the signal by predicting the source’s position, and transmits/receives messages processed in an FPGA.

My Role / Contributions:

  • Designed and routed portions of a 5-board modular PCB stack in KiCad around an Arduino GIGA R1 WiFi, separating sensor, intermediary, and control boards for better scalability and debugging.
  • Built a custom multi-channel PCB with multiplexers for automated electrode switching, eliminating manual reconfiguration.
  • Performed DRC checks and validated PCB manufacturability, coordinating fabrication to ensure successful production.
  • Built and tested sensing hardware, collecting resistivity data and correlating measurements to subsurface material properties.
